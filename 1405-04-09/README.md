### Project 1

یک برنامه بنویس که:

1. نام کاربری GitHub را از کاربر دریافت کند.
2. به API گیت‌هاب درخواست GET بفرستد.
3. اگر کاربر وجود داشت، اطلاعات زیر را نمایش دهد:
   - Login
   - Name
   - Bio
   - Followers
   - Following
   - Public Repositories
   - Created At
4. اگر کاربر وجود نداشت، پیام مناسبی نمایش دهد.
5. خطاهای احتمالی (مثل قطع اینترنت) را مدیریت کند.

### Project 2

یک برنامه بنویس که:

1. آدرس یک سایت را از کاربر بگیرد.
2. با requests به آن درخواست GET بفرستد.
3. اگر سایت در دسترس بود بنویسد:
   Website is Online
4. اگر در دسترس نبود بنویسد:
   Website is Offline
5. Status Code را نیز چاپ کند.
6. خطاهای ConnectionError و Timeout را مدیریت کند.
